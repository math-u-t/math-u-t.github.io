GoogleAppsScriptのフレームワーク[Linkgs]

2025.08.06

GoogleAppsScriptに新しい機能「ページ遷移」を容易に実装できるフレームワーク「Linkgs」を作成しました！

GoogleAppsScriptのフレームワーク[Linkgs]

通常のようにhref属性を使用できます。

function returnHTML(templateObj, file, title) { const now = new Date(); const mainTemplate = HtmlService.createTemplateFromFile(file); for (const key in templateObj) { mainTemplate[key] = templateObj[key]; } const linkTemplate = HtmlService.createTemplateFromFile("link.js"); for (const key in templateObj) { linkTemplate[key] = templateObj[key]; } const mainHtml = mainTemplate.evaluate().getContent(); const linkHtml = linkTemplate.evaluate().getContent(); const finalHtml = mainHtml + linkHtml; const output = HtmlService.createHtmlOutput(finalHtml) .setTitle(title) .setXFrameOptionsMode(HtmlService.XFrameOptionsMode.ALLOWALL) .setFaviconUrl(sc_props("favicon")) .addMetaTag("viewport", "width=device-width, initial-scale=1") .addMetaTag("google-site-verification", "spirix | " + now); return output; }

これにより、GoogleAppsScriptに容易にページ遷移を実装できるようになりました。
